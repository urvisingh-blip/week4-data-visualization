# week4-data-visualization
# Data Visualization & Complete Analysis Project

## Project Overview
Complete data analysis project with visualizations using
the sales dataset. Created 3 different chart types to
analyze sales patterns.
Built as Week 4 Final Project for The Developers Arena
Data Science Internship.

## Setup Instructions
1. Open Google Colab (colab.research.google.com)
2. Upload sales_analysis_week4.ipynb
3. Run the cell (Shift + Enter)
4. Upload sales_data.csv when prompted

## Code Structure
- Step 1: Load data using pandas
- Step 2: Clean data - handle missing values
- Step 3: Analyze - revenue, products, regions
- Step 4: Chart 1 - Bar chart (Sales by Product)
- Step 5: Chart 2 - Pie chart (Sales by Region)
- Step 6: Chart 3 - Horizontal bar (Quantity Sold)
- Step 7: Final report with insights

## Charts Created
- Bar Chart: Total Sales by Product
- Pie Chart: Sales Distribution by Region
- Horizontal Bar Chart: Quantity Sold by Product

## Key Insights
- Total Revenue: Rs.12,365,048
- Best Selling Product: Laptop
- Top Region: North
- Laptop and Tablet together contribute majority of revenue
- North and South regions dominate sales
